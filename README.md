## About Me

I am a cybersecurity professional with a strong focus on vulnerability management, exposure analysis, asset visibility, technical validation, and security operations.

My work is mainly related to identifying, analyzing, prioritizing, and communicating security risks across infrastructure, endpoints, cloud assets, and exposed services.

I also build scripts and small tools to automate repetitive security tasks, improve reporting, process technical data, and make security analysis more efficient.

Alongside my defensive security background, I am currently expanding my knowledge in Offensive Security, with interest in pentesting, exploitation concepts, Active Directory security, Linux security, and hands-on labs.
